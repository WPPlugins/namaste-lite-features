== Description ==

Namaste Lite Features is a WordPress plugin for Namaste Lite theme from Webzakt.
Get free or pro Namaste WordPress Theme from webzakt.com.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/namaste-lite-features` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
